# Installation

Prérequis :
NodeJS
npm package manager

1. Installer le cli d'angular
```
npm install -g @angular/cli
```
2. Installer les dépendances du projet
```
cd angular-chat
npm install
```

## Lancement

Executer `ng serve` puis rendez-vous sur `http://localhost:4200/`
